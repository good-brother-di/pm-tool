# PM Tool 项目管理软件 — 用户手册

## 1. 软件概述

PM Tool 是一款基于 Web 的项目管理看板软件，支持项目管理、任务看板、甘特图、活动日志、AI 周报生成、数据导出等功能。采用 Next.js 全栈框架，SQLite/Turso 云数据库。

## 2. 运行环境

- 操作系统：Windows / macOS / Linux
- 浏览器：Chrome / Edge / Firefox 最新版
- 服务端：Node.js 18+

## 3. 安装步骤

3.1 安装 Node.js（https://nodejs.org，下载 LTS 版本）
3.2 解压软件包到任意目录
3.3 双击 start.bat 启动服务器
3.4 浏览器打开 http://localhost:3000

## 4. 功能说明

### 4.1 仪表盘
- 首页展示项目总览、任务统计饼图、过期/阻塞任务列表
- 统计卡片显示总任务数、待办数、阻塞数、完成率

### 4.2 项目管理
- 点击左侧「新建项目」创建项目
- 侧边栏拖拽调整项目顺序
- 支持项目删除（含确认提示）

### 4.3 任务看板
- 四列看板：待办、进行中、阻塞、已完成
- 拖拽任务卡片切换状态
- 同列内拖拽排序
- 点击任务卡片打开详情编辑面板
- 支持标题、描述、优先级、截止日期、标签

### 4.4 甘特图
- 时间线视图，显示任务色条
- 月份头部、今日红线标记
- 按任务状态着色
- 支持横向滚动

### 4.5 活动日志
- 自动记录任务创建、移动、删除等操作
- 显示操作时间和详情
- 按项目筛选

### 4.6 AI 周报
- 调用 DeepSeek API 自动生成中文周报
- 包含概览、关键进展、风险与阻塞、下周计划

### 4.7 数据导出
- 支持 CSV 和 Markdown 格式导出
- 一键下载项目所有任务数据

### 4.8 深色模式
- 侧边栏底部切换浅色/深色主题
- 所有组件完整适配

### 4.9 Obsidian 同步
- 一键导出项目数据到 Obsidian 知识库

## 5. 技术架构

- 前端：Next.js 16 + React 19 + TypeScript
- UI 组件：shadcn/ui + Tailwind CSS
- 拖拽：@dnd-kit
- 数据库：Prisma ORM + Turso/libsql
- 图表：Recharts
- AI：DeepSeek API

## 6. 快捷键

- 点击任务卡片：打开详情编辑面板
- 拖拽任务：切换状态 / 排序
- 拖拽项目：调整侧边栏顺序
