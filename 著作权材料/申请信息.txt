=== 软件著作权申请表 — 信息汇总 ===

软件全称：PM Tool 项目管理软件
简称：PM Tool
版本号：V1.0

- 软件分类：应用软件 - 通用办公软件
- 开发方式：独立开发
- 开发完成日期：2026年6月
- 首次发表日期：2026年6月

- 硬件环境：PC，2GB+ RAM
- 软件环境：Windows/macOS/Linux + Node.js 18+ + 浏览器
- 编程语言：TypeScript (4156行)

- 申请人：严能
- 证件类型：身份证

- 主要功能：
  1. 项目看板管理（拖拽排序、状态切换）
  2. 甘特图时间线视图
  3. 活动日志自动记录
  4. AI 周报生成
  5. 数据导出（CSV/Markdown）
  6. Obsidian 知识库同步
  7. 深色模式

- 技术特点：
  1. 基于 Next.js 16 全栈框架，SSR 服务端渲染
  2. @dnd-kit 拖拽引擎，支持看板跨列移动和同列排序
  3. Prisma 7 ORM + Turso 分布式云数据库
  4. shadcn/ui 组件库 + Tailwind CSS 响应式布局
  5. 集成 DeepSeek AI 大模型自动生成项目周报
